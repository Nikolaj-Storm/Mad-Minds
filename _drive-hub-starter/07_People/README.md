# 07_People — personal workspaces

One subfolder per marketer (lowercase first name, e.g. `nikolaj/`). This is where your drafts, work-in-progress, and experiments live.

- Skills save here **by default**.
- When something is finished and team-relevant, say **"publish to the team"** and the skill copies the final file into the shared folder (`04_Reports/`, `05_Plans_and_Strategy/`, or `03_Data/cleaned/`) with house naming. The working copy stays here.
- Every `@onlineminds.io` account has Editor access to the entire Hub. Drive's per-file version history (File → Version history) is the safety net if something is overwritten by accident.
